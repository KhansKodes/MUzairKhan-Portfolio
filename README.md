# Muhammad Uzair Khan - Portfolio

A modern, responsive portfolio website built with React and Vite.

## Features

- Modern React architecture with component-based structure
- Smooth animations and transitions
- Responsive design for all devices
- Neural network background animation
- Professional sections: About, Resume, Experience, Education, Portfolio, Contact

## Tech Stack

- React 18
- Vite
- Bootstrap 5
- AOS (Animate On Scroll)
- Custom CSS with modern design

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Component Structure

Each component has its own folder containing:
- `ComponentName.jsx` - Component logic
- `ComponentName.css` - Component styles (if applicable)
- `index.js` - Export file for cleaner imports
